[SECTION: CONTROL_PANEL_AND_LIGHTS]
Button Layout:
- Power Button (Back of printer)
- Wi-Fi Reset Button (Back of printer)
- Information Button (i): Prints information page
- Resume/Cancel Button (X): Resumes job or cancels current operation
- Copy Buttons: Black Copy (right), Color Copy (left)

Edge Lighting Status Codes:
1. Pulse Purple: Wireless setup mode (ready to connect).
2. Pulse Blue: Connecting to Wi-Fi.
3. Solid Blue: Connected to Wi-Fi/Ready.
4. Solid Amber (Large Bar): Printer warning or blocking error (check paper/ink).
5. Fast Blinking Amber: Serious error (requires restart).
6. Pulse White: Printing in progress.
7. Green Glow (Center): Updating firmware.
8. Rotating Blue: Wi-Fi Direct is active.

[SECTION: SETUP_INSTRUCTIONS]
Process:
1. Remove all tape and cardboard from the printer (check inside cartridge access door).
2. Plug into power source. The printer turns on automatically.
3. Wait for the printer lighting to turn Purple (Setup Mode).
4. Download the "HP Smart" app on a mobile device or PC (123.hp.com).
5. Open HP Smart and click "Add Printer" (+).
6. Follow on-screen prompts to connect printer to Wi-Fi.
7. Install ink cartridges when prompted by the app.
8. Load paper and perform alignment.

Wi-Fi Reset (If printer is not found):
- Locate the Wi-Fi button on the back of the printer.
- Press and hold for at least 5 seconds until the Edge Lighting pulses purple.

[SECTION: TROUBLESHOOTING_COMMON_ERRORS]
Error Code E0: Unusable Cartridge. 
- Fix: Reseat cartridges, clean contacts, or replace.
Error Code E1 / E2: Paper Size Mismatch.
- Fix: Ensure paper size settings in software match the paper loaded.
Error Code E3: Paper Jam (Carriage Jam).
- Fix: Open cartridge access door, clear jammed paper from the path, close door, press Resume.
Error Code E4: Paper Jam (Input).
- Fix: Check input tray for jammed paper.
Error Code E9: Firmware Update Error.
- Fix: Restart printer and retry update via HP Smart app.

[SECTION: MAINTENANCE]
Cleaning Printhead:
- Perform via HP Smart App > Printer Settings > Print Quality Tools > Clean Printhead.
Aligning Printhead:
- Perform via HP Smart App > Printer Settings > Print Quality Tools > Align Printhead.
Scanner Cleaning:
- Use a soft, lint-free cloth and glass cleaner on the scanner glass to remove streaks.

[SECTION: FAQ]
Q: Does the HP Envy 6000 scan double-sided automatically?
A: No, the ADF (if equipped on specific sub-models like 6400) is usually simplex. The 6000 series typically uses a flatbed scanner and does not support automatic duplex scanning, only manual.

Q: How do I print a test page?
A: Press and hold the Information (i) button and the Resume (X) button simultaneously for 3 seconds.

Q: Why is my printer offline?
A: Check if the Edge Light is blue. If not, press the Wi-Fi button on the back to reconnect. Restart the router and printer.
